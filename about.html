About

Original project

This site is a reformatted version of the original Shirokuma Cafe Japanese learning project created by George Matthews. All episode transcripts, audio clips, and English translations are his work. This version adds a data-driven architecture (JSON episodes, single template), improved navigation, and extends the project to all 50 episodes.

The original project is available at gwmatthews.github.io/Shirokuma-Cafe and on GitHub.

What is this?

Each episode page presents the Japanese dialogue scene by scene with an audio clip for each scene — click 🔊 to hear it. English translations are hidden by default; hover over a line to reveal it, or press the EN button to show all translations at once.

You can explore all 50 episodes at shirokuma.ouaic.com.

Beyond Shirokuma Cafe

The ideas behind this project — parsing subtitles, clipping dialogue audio, and generating translations — are repurposeable. In theory the same approach works for any language, any show, as long as subtitle files exist. Each new show would be its own dedicated project, built and tuned for that content.

The translation pipeline uses Claude Haiku by default but can be swapped for any model or API — the code is open, self-hostable, and designed to be tweaked. In future, a language-agnostic version of this tooling may be published on GitHub for anyone looking to build their own.
