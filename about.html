About

Original project

This site is a reformatted version of the original Shirokuma Cafe Japanese learning project created by George Matthews. All episode transcripts, audio clips, and English translations are his work. This version adds a data-driven architecture, improved navigation, and extends the project to all 50 episodes.

The original project is available at gwmatthews.github.io/Shirokuma-Cafe and on GitHub.

What is this?

Each episode presents the Japanese dialogue scene by scene with an audio clip per scene. Click 🔊 to hear it. English translations are hidden by default, hover over a line to reveal it, or hit the EN button to show them all at once.

Have a browse at shirokuma.ouaic.com.

Beyond Shirokuma Cafe

The ideas here are repurposeable. Parsing subtitles, clipping audio, and generating translations can in theory be applied to any show in any language as long as subtitle files exist. Each show would be its own project, tuned to its content.

The translation pipeline uses Claude Haiku by default but you can swap in any model or API, self-host the whole thing, and tweak the code however you like. Down the line a more language-agnostic version may be published on GitHub for anyone who wants to do something similar.
