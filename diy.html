Build Your Own

Want to create a site like this for a different show? Here's the full pipeline.

Prerequisites

You'll need:

  • Video files for the show (.mkv or similar)
  • Japanese subtitle files (.ass or .srt) — try kitsunekko.net or jimaku.cc
  • Docker (for ffmpeg audio clipping)
  • Python 3 + an Anthropic API key (for translation)

Step 1 — Clone the repo

git clone https://github.com/ouaickaka/shirokuma-cafek-zoku
cd shirokuma-cafek-zoku
cp env.template .env
# Add your ANTHROPIC_API_KEY to .env
pip install -r requirements.txt

Step 2 — Build an episode

Run build-episode.py with the episode number, subtitle file, and video file:

python3 build-episode.py 1 \
  "subtitles/MyShow-01.ja.ass" \
  "/media/MyShow/MyShow-S01E01.mkv"

This will:

  • Parse the subtitle file and filter out noise (sound effects, credits)
  • Group lines into scenes (~3 lines per scene)
  • Clip one MP3 per scene using ffmpeg
  • Write episodes/01.json with all dialogue

Step 3 — Translate

python3 translate.py episodes/01.json en

This calls Claude Haiku to translate each Japanese line in-place. Already-translated lines are skipped, so you can re-run safely.

Step 4 — Rebuild the index

python3 convert-to-json.py 0 0  # rebuilds episodes/index.json

Step 5 — Serve

docker compose up -d

The site is now live at http://localhost:8080.

Customising

Edit includes/style.css for styling. The episode template is episode.html — it's plain vanilla JS with no dependencies.
